# Data Description

Synthetic dataset of 1000 short messages.

**File:** `data/sms_sender_dataset_1000.csv`

## Columns
- `text`
- `time_of_day` in {morning, afternoon, evening, night}
- `label` in {Mom, Boss, Alex, Friend}

## Class distribution
| label   |   count |
|:--------|--------:|
| Friend  |     289 |
| Mom     |     258 |
| Boss    |     233 |
| Alex    |     220 |
